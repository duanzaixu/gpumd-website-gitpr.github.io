最新稳定版本 GPUMD-v4.9.1

GPUMD-v4.9.1 亮点

Bug 修复与功能增强

立即下载 v4.9.1

nep 可执行文件：

  • 对于缺失应力参考值结构，其目标应力的输出值统一修正为 -1e6。此前该值为较大的负数，但数值不固定。 #1364

gpumd 可执行文件：

  • 修复了 v4.9 引入的一个 Bug。当使用单 GPU 运行 NEP 或 qNEP 模型，且周期性盒子的厚度处于 2.5rc 与 2.5(rc + 1) 之间时，部分邻居原子会被重复计算。 #1370
  • 修复了 ensemble ti_rs 关键词的一个 Bug。解决了当 tswitch 与 tequil 的乘积超过整型最大值 INT_MAX (231-1) 时导致的溢出问题。 #1366

快速开始

仅需三步，即可在您的环境中部署 GPUMD。

Step 01

环境准备

加载 CUDA 编译环境。

module load cuda
Step 02

解压源码

解压下载的 ZIP 包。

unzip GPUMD.zip
Step 03

编译程序

进入 src 目录并执行 make 编译。

cd GPUMD/src && make

GPUMD-v4.9.1 Highlights

Bug Fixes and Enhancements

Download v4.9.1 Now

nep executable:

  • For missing stress reference structures, the target stress output is unified to -1e6. Previously this was a large negative number, but not fixed. #1364

gpumd executable:

  • Fixed a bug introduced in v4.9. When running NEP or qNEP models with a single GPU, and the periodic box thickness is between 2.5rc and 2.5(rc + 1), some neighbor atoms were double counted. #1370
  • Fixed a bug in the ensemble ti_rs keyword. Solved an overflow issue when the product of tswitch and tequil exceeds the maximum integer value INT_MAX (231-1). #1366

Quick Start

Deploy GPUMD in your environment in just three steps.

Step 01

Prepare Env

Load the CUDA compilation environment.

module load cuda
Step 02

Unzip Source

Unzip the downloaded ZIP package.

unzip GPUMD.zip
Step 03

Compile

Enter src directory and run make.

cd GPUMD/src && make
